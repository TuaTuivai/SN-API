# SN-API


## Description
This is an API that powers a social network website where users can share their ideas, respond to their friends' thoughts, and create a list of friends. It allows users to interact with one another by posting their opinions and reacting to others' posts. Additionally, they can connect with other users by creating a list of friends they follow.

## Installation
To run this project, install it locally using 

* npm install

* Mongoose
* Express.js

## Usage
After installing npm packages, the application can be used by using the following command:

npm start
